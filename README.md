# hadiubt

Personal website of Umut Barış Terzioğlu, QA Engineer based in Dortmund, Germany.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- React Router DOM

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
